# Contributing to ZK-Neura dApp

Thank you for your interest in contributing to ZK-Neura.

## Principles

- Privacy first
- Type-safe by default
- Small, composable components
- Clear, maintainable code

## Tech Stack

- Next.js 16 (App Router)
- React 19
- TypeScript
- TailwindCSS v4
- shadcn/ui
- Zustand
- Solana Web3.js
- WebCrypto API

## Development Setup

1. Fork this repository.
2. Clone your fork:
   ```bash
   git clone https://github.com/<your-username>/zkneura-dapp
   cd zkneura-dapp
3. Install dependencies:
   ```bash
   npm install
4. Create env file:
   ```bash
   cp .env.example .env.local
5. Start development:
   ```bash
   npm run dev
